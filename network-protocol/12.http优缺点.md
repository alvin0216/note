## HTTP 优点

1. 灵活可扩展：不仅仅可以传输文本，还能传输图片、视频等任意数据，非常方便。
2. 可靠传输：HTTP 基于 TCP/IP，因此把这一特性继承了下来。这属于 TCP 的特性，不具体介绍了。
3. 请求-应答：也就是一发一收、有来有回，
4. 无状态：这服务器没有状态差异，可以很容易地组成集群。

## HTTP 缺点

1. 无状态：无法记录请求的对象信息，为此 cookie 存在的意义。。
2. 明文传输：容易被抓包，窃取、篡改、冒充。
