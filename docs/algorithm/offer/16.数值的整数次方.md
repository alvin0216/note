实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。不得使用库函数，同时不需要考虑大数问题。

```js
/**
 * @param {number} x
 * @param {number} n
 * @return {number}
 */
var myPow = function (x, n) {
  if (n === 0) return 1; // 所有数的0次幂都是1
  // n小于0，特殊情况
  if (n < 0) return 1 / myPow(x, -n);

  // 奇数
  if (n % 2) return x * myPow(x, n - 1);
  // 偶数
  return myPow(x * x, n / 2);
};
```

n 偶数情况

比如 x=2 n=4

`2 * 2` 一轮之后 n = 2, n 是对半的

n 奇数情况

比如 x=2 n=5

可以拆分 `2 * 2的四次方` 以此类推

注意 n < 0 和 n = 1 情况就是了。
