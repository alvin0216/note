## Typescipt 汇总

- 类型运算
  - 集合运算
  - 索引签名
  - 类型键入
  - typeof value 获取属性类型
  - keyof Type 获取 key 值
  - extends
- 泛型
  - 1⃣️ 映射类型
    - 关键字 in
    - `Record<K,T>` 属性 K 定义为类型 T
    - Partial 可选值
    - Required 必选值
    - Readonly 只读
    - Pick 属性提取
    - Omit 属性排除
  - 2⃣️ 条件类型
    - 三目运算符
    - info 动态推导
    - ReturnType 获取返回值类型
    - Parameters 获取方法的参数类型
    - Exclude 属性排除
    - Extract 属性交集
    - NonNullable 排除 null 和 undefined
